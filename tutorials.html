C++ Tutorial 01

To start out with C++, write the following code:
#include <iostream>

int main()
{
  std::cout << "Hello World" << std::endl;
}
The #include tells our program that we want to use the tools within the file specified. std::cout tells our program that we want to print something out to the console, using << tells the program that we are outputting information and not taking it in. You want whatever you are printing out to be in "" or '' (unless it is a variable). std::endl tells our program that we want to make a new line.

Variables

In C++ we have variables. Variables are important because they allow us to work with numbers and make complex calculations. I may not know all of the variables, but I know the main ones. The first variable you have is an int, it is defined like this:
int i = 10;
In this example, we have set i to 10. In C++ it is important to define variables like this, even if you set the variable to be 0. The reason for this is because C++ is heavy on memory managment. So you don't want to be creating variables with no value. If you set the variable to NULL, that is the same thing as not setting it, you can also set the variable after defining it, but you should do this before using the variable. You can also define variables pretty much anywhere in your code.
Another variable is a string, it is defined like this:
